# 🏥 Медицинская исследовательская платформа

Веб-приложение для медицинских исследований (проект для диссертации).  
Работает локально на вашем компьютере.

## 📋 Описание проекта

Это каркас медицинской платформы с двумя типами пользователей:
- **Пациенты** - могут войти в личный кабинет
- **Врачи** - могут войти в панель управления

**Текущий этап:** Базовый каркас приложения (без реальной авторизации и базы данных).

## 🛠 Технологии

- **Бэкенд:** Python + FastAPI
- **Фронтенд:** HTML + CSS + JavaScript
- **База данных:** SQLite (будет добавлена на следующих этапах)

## 🚀 Как запустить проект

### Шаг 1: Проверьте наличие Python

Откройте командную строку (cmd) и введите:

```bash
python --version
```

Должна отобразиться версия Python 3.8 или выше. Если Python не установлен, скачайте его с [python.org](https://www.python.org/downloads/).

### Шаг 2: Установите зависимости

В командной строке перейдите в папку проекта:

```bash
cd C:\Users\Anastasia\Desktop\mybot
```

Установите необходимые библиотеки:

```bash
pip install -r requirements.txt
```

### Шаг 3: Запустите сервер

Запустите FastAPI сервер командой:

```bash
python backend/main.py
```

Вы увидите сообщение:
```
INFO:     Uvicorn running on http://127.0.0.1:8000
```

### Шаг 4: Откройте сайт в браузере

Откройте браузер и перейдите по адресу:

```
http://127.0.0.1:8000
```

Или просто:

```
http://localhost:8000
```

## ❓ Возможные проблемы и решения

### Ошибка: "pip не является внутренней командой"

**Решение:** Переустановите Python и убедитесь, что отметили галочку "Add Python to PATH" при установке.

### Ошибка: "Address already in use"

**Решение:** Порт 8000 уже занят. Измените порт в файле `backend/main.py`:
```python
uvicorn.run(app, host="127.0.0.1", port=8001)  # Используйте другой порт
```

### Страница не загружается

**Решение:** 
1. Убедитесь, что сервер запущен (смотрите в командную строку)
2. Проверьте правильность адреса: `http://127.0.0.1:8000`
3. Попробуйте другой браузер
